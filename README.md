# Aryan Sharma — Portfolio

> Personal portfolio website hosted on GitHub Pages

---

## About

Product Manager with ~5 years of experience delivering scalable, API-driven platforms across FinTech, E-commerce, Telecom, and EdTech. This portfolio showcases work history, skills, and projects.

---

## Features

- Dark / Light mode toggle with localStorage persistence
- Particle background (desktop only — auto-disabled on mobile for performance)
- Typed.js hero animation with product-focused strings
- AOS scroll-reveal animations
- Work History modal with full timeline (dates and bullets from resume)
- Resume modal with inline PDF viewer + download (`ARYAN_SHARMA_PRODUCT.pdf`)
- Education & Certifications section
- Testimonials carousel
- Contact form (Netlify-ready)
- Fully responsive — mobile, tablet, desktop

## Deployment (GitHub Pages)

To push updates live:

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

GitHub Pages automatically rebuilds from the `main` branch.
